#HR Dashboard

## Project Overview
This project involves building an interactive Power BI dashboard to analyze employee demographics. The project utilizes multiple raw CSV datasets to build a relational data model and extract actionable insights.

## Data Modeling & Architecture
This project features full **CSV Data Modeling**. Seperate Source tables were imported and connected inside Power BI using a "Snowflake Schema" layout to establish relationships via primary and foreign keys.

## Tools & Tech Stack
* **Power BI Desktop:** For data modeling and dashboard creation.
* **Power Query:** For data transformation and cleaning (ETL).
* **DAX (Data Analysis Expressions):** Used to write custom measures for deep-dive metrics.

## Repository Contents
* 'Dashboard.pbix' - The core Power BI project file.
* 'Attendance.csv', 'Departments.csv', 'Employees.csv', 'Projects.csv' - The source datasets used to build the data model.
